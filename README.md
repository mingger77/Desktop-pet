# Desktop Pet 桌面宠物

基于 PySide6 的桌面宠物应用。无边框透明置顶窗口，使用 PNG 精灵图展示女仆角色，通过状态机驱动交互和动画。

## 功能介绍

### 桌面宠物

桌面上有一只女仆形象的小宠物，窗口无边框、可穿透点击、始终置顶显示。你可以随意拖动她到桌面的任何位置，她会在原地待机，偶尔轻轻浮动。

### 点击互动

- **左键单击**宠物：她会根据当前状态做出随机反应。待机时通常是可爱或受惊的表情，懒散时会道歉。每次反应持续约 1.5 秒后自动恢复。
- **拖拽**宠物：按住左键拖动即可移动她的位置。
- **右键单击**：弹出功能菜单。

### 空闲状态

如果你连续 10 分钟没有点击宠物，她会切换到懒散状态，表现出慢悠悠的浮动动画。点击她即可唤醒。

### 石头剪刀布

在右键菜单中选择「游戏」，宠物会邀请你玩石头剪刀布。

- 弹出粉色主题的选择面板，点击石头、剪刀或布
- 宠物会随机出拳，展示 1.5 秒
- 根据胜负结果，宠物会展示胜利、失败或平局的表情，持续 2 秒
- 对话框不会自动关闭，你可以连续多轮游玩
- 点击「退出游戏」可随时结束

### 跳舞

在右键菜单中选择「跳舞」，会弹出跳舞模式选择窗口，提供三种模式：

**模式 A：随机跳舞**
宠物从 6 个舞蹈动作中随机选择 4 个，循环 3 次，给你一段即兴表演。

**模式 B：按指示跳舞**
显示你之前保存的自定义舞蹈列表，翻页式浏览，每个舞蹈显示循环次数。选择其中一个即可播放。最多可保存 3 组舞蹈。

**模式 C：自定义编舞**
你可以从 6 个舞蹈动作中自由勾选 3~6 个，选择循环次数（2~4 次），然后播放或保存到列表。

> 注：跳舞功能需要存在 `maid--dance1.png` ~ `maid--dance6.png` 六张精灵图。

### AI 聊天

在右键菜单中选择「工作」，宠物会化身为 AI 助手「女仆酱」。

- **首次使用**：需要配置 API Key 和 Base URL（支持 OpenAI 兼容接口），以及选择模型名称。配置一次后永久保存。
- **配置好后**：可以选择「开始聊天」进入对话，或「修改配置」更改 API 信息。
- **聊天界面**：宠物切换到工作状态，弹出粉色调的聊天窗口。布局如下：

```
[输入框                           ] [发送]
[工作模式] [联网模式] [位置配置] [关闭]
```

- **工作模式**：开关按钮，开启后 API 请求携带文档生成工具，AI 可自动调用生成 PPT/Word/Excel/PDF。
- **联网模式**：开关按钮，开启后 API 请求携带联网搜索参数。
- **位置配置**：设置文件输出目录，默认桌面。选择后路径保存到 `config.json`，下次启动自动加载。桌面不可用时自动弹出文件夹选择器。
- **高级设置**（旧版弹窗方式已移除，改为内联按钮）。

### 配置对话框

- **API Key 可见性切换**：输入框旁有「显示」按钮，点击可查看完整密钥，再次点击恢复掩码显示（前 6 位 + ******）。对话框关闭后自动恢复掩码。
- **教程入口**：配置对话框底部有蓝色「不懂点击这里」链接，点击后弹出 24 页 DeepSeek API 配置教程。

### 文档生成

工作模式开启后，AI 可通过函数调用（function calling）自动生成文档，支持 PPT、Word、Excel、PDF 四种格式。生成过程在后台执行，不阻塞聊天。

### 退出

右键菜单中选择「退出」即可关闭程序。
